发表于 2017-03-06

介绍

yarn是Facebook发布的Node.js包管理器，旨在取代npm。官方介绍yarn为快速、可靠、安全的依赖管理器

安装

yarn有多种安装方式，下面介绍两种安装方式(windows环境下)：

  • 下载msi文件安装

    • 下载地址： https://yarnpkg.com/latest.msi

  • npm 的方式安装

上述命令会自动将[packgae]添加到项目依赖中,相当于npm install --save [package]

如果要添加其他依赖类型，可以使用如下命令

  • yarn add --dev 添加到 devDependencies
  • yarn add --peer 添加到 peerDependencies
  • yarn add --optional 添加到 optionalDependencies

安装依赖项

  • 安装所有依赖：yarn 或 yarn install
  • 安装一个包的单一版本：yarn install --flat
  • 强制重新下载所有包：yarn install --force
  • 只安装生产环境依赖：yarn install --production
